1.检索功能使用说明：【仅返回文字描述信息】

    1）在“关键词表.xlsx”文件中填写关键词。不要做列名、表名改动。

    2）“关键词表.xlsx”的limit列代表检索保留结果数量。建议写个数字，新华社资源多，保留太多无必要。且会影响检索速度

    3）检索结果在data文件夹下。请及时改名，以免下次运行覆盖。

    4）针对3种媒体格式，运行query-xxx.bat文件。


2.缩略图下载功能使用说明：【根据图片id下载相应缩略图】

    1）在“图片id列表.xlsx”文件中填写图片的id。不要做列名、表名改动。

    2） 执行downloadIMG.bat文件。

    3）下载图片在data文件夹下的以时间为前缀的目录下。同时，data文件夹下有一个说明excel文件，收集了图片的说明、标签等信息。


3.检索关键字、将对应结果的文字描述及缩略图插入excel中：【最新版本，推荐使用】

    1）在“关键词表.xlsx”文件中填写关键词。不要做列名、表名改动。

    2）执行glue-精确检索下载插入图片.bat。

    3）下载图片在/data/image目录下；/data/检索结果.xlsx，收集了图片图片的说明、标签等信息以及缩略图。

    4）建议检索分批进行，每批的检索量（检索词个数×保留结果个数）控制在1w以内。
